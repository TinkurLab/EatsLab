## Grind Sizes 
Using a Baratza Virtuoso Burr Grinder

| Method | Grind Size |
| - | - |
| espresso | 2-6 |
| drip | 13-14|
| Chemex | 20-21 (25 or higher for darker roasts) |
| AeroPress | 8-11 |
| French Press | 30-32 | 

## Water Ratios 

| Water to Coffee Ratio | Result |
| - | - |
| 10:1 | strong can't see through it coffee |
| 15: 1 | medium semi transparent coffee |
